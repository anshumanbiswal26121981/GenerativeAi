# Revision Notes: Introduction to AI and LLMs

## Overview
This session serves as an introductory guide to understanding the basic vocabulary and concepts in the realm of Artificial Intelligence (AI), with a particular focus on Large Language Models (LLMs).

The objective is to familiarize you with key terms and concepts that will be foundational for future classes.

# AI Vocabulary

## AI (Artificial Intelligence)
AI is a broad term for computer systems that can perform tasks that require human-like intelligence.

These tasks include:
- Image recognition
- Natural language processing
- Decision-making
- Playing games like chess

---

## Narrow vs. General AI

### Narrow AI
What exists today; designed to handle one specific task.

Examples:
- Spam filters
- Face recognition

### General AI
A hypothetical AI that could perform any intellectual task a human can do.

---

## Generative AI
Generative AI refers to systems that create new content.

While traditional AI techniques classify and predict, generative AI can:
- Write
- Draw
- Compose music
- Generate code

---

## LLM (Large Language Model)
LLMs are a subset of AI and products of deep learning within machine learning.

They are capable of generating human-like text by predicting the next word in a sentence.

---

## AI Agents
An AI Agent extends beyond simple chat responses by performing tasks autonomously.

Examples include:
- Booking flights
- Making reservations

Capabilities:
- Uses tools
- Retains memory of past interactions
- Makes autonomous decisions

---

# Core Concepts in LLMs

## Tokens
Tokens are the smallest units of text that an LLM processes.

Think of them like LEGO bricks for text.

A token can be:
- Part of a word
- A punctuation mark
- A space

### Importance of Tokens

#### Billing
API usage is typically metered by the number of tokens processed.

#### Context Limits
Models have a limited number of tokens they can process at one time.

---

## Context Window
A context window refers to the amount of information (in tokens) an LLM can handle at once.

Analogy:
- Small context window → small desk
- Large context window → large conference table

---

## Temperature
Temperature is a parameter in LLMs that controls randomness in predictions.

- **Lower temperature** → More deterministic and focused outputs
- **Higher temperature** → More varied and creative responses

## JSON and Terminal

### JSON (JavaScript Object Notation)
A data interchange format widely used in APIs for structuring data in a readable way.

### Terminal
A command-line interface used to interact with the operating system.

Essential for:
- Navigating file systems
- Running scripts
- Managing development workflows
